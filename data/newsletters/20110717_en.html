Piwigo 2.2.4

In this new stable version, we have added a new language Bulgarian, thanks to hamsia, josifbg and n0kS. This new version contains many changes and bug fixes, such as:

  • improved CSS file selector for plugin LocalFiles Editor: simpler listbox and default theme CSS custom file is automatically loaded



  • when you create a sub-album in a private album, admins and album creator automatically get permission on it (but you can remove this permission afterwards).

  • new methods in Piwigo API to set a photo as representative of an album or to set photos sort order in an album

  • to increase performances, Piwigo 2.2.0 has removed a bit of randomness on thumbnails when an album has no defined representant. With Piwigo 2.2.4, you can get this randomness back with local configuration parameter $conf['representative_cache_on_subcats'] = false

  • the main problem users encountered during migration to Piwigo 2.2 was that database upgrade was not performed correctly. Piwigo 2.2.4 will detect this missing upgrade and launch it automatically.

Piwigo 2.2.4 has many more changes, please read them all on Piwigo 2.2.4 Release Notes.

Follow the automatic upgrade procedure and run the most recent version of Piwigo.

Here is the list of changes compared to Piwigo 2.3.0beta1:

  • [feature 2317] move order config to Admin->Config->Options

  • [feature 2372] [LocalFiles Editor] shortcut to edit CSS file from the themes page

Showcase on Piwigo.org

The Piwigo Showcase is under preparation. We can also name it "Live Piwigo Galleries Directory". Actually the Piwigo Showcase is a Piwigo gallery itself, where each Piwigo live gallery is represented as a single screenshot, with a date of birth, a description, tags and URL.



A new plugin is available Showcase Register : install and activate it on your gallery. Then fill the form to submit your registration on the Showcase.

The form is very easy to use, all mandatory fields are prefilled. Don't hesitate to fill the optional fields as well, it will help Piwigo Showcase administrators to describe and tag your gallery.



The plugin is compatible with Piwigo 2.1+. If you have an older version of Piwigo and want to register, reply to this email and give us your URL and a description of your gallery.

Once your registration is submitted, an administrator of the Piwigo Showcase will verify it, make a screenshot of your gallery and validate your registration. You will then be notified by email, if you have provided your email address.
