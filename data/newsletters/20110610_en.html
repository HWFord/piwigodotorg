Hello Piwigo user, you are receiving this newsletter because you have signed up to receive Piwigo emails. Piwigo is the Free Open Source photo gallery software for the web.

Piwigo 2.2.2

Release 2.2.2 is available. Language Greek (Ελληνικά) was added and many other languages were updated. All details available on Piwigo 2.2.2 release notes

  • due to many bugs on the tags manager, we have replaced the technical library. It is visually equal, but all bugs should be fixed now



  • on the Batch Manager, in unit mode, you can zoom on each photo with nice lightbox (easier to tag photos when you can see a picture picture than a thumbnail)

  • if you use FTP + synchronization to add photos and if you have many albums (I mean thousands of albums), you will appreciate the new "synchronize" button on the album administration screen : no need to find it in a select list on the synchronization screen



  • web API gets new options, pwg.categories.getList is able to return a tree instead of a flat list of albums and pwg.categories.getImages now returns the creation and posting date of each photo (for higher speed in the Lightroom export plugin)

Follow the automatic upgrade procedure and run the most recent version of Piwigo.

Piwigo 2.2.3

Release 2.2.3 was released a few days after release 2.2.2. It mainly fixes a few bugs introduced in 2.2.2, all details available on Piwigo 2.2.3 Release Notes.

Piwigo 2.3.0beta1

Here comes version 2.3.0beta1. This version is for test only, so don't install it on your live website. Even if Piwigo 2.2 was released only a few weeks ago, we already have interesting new features to show for the future Piwigo 2.3.

1) Changes

Here is the list of changes compared to Piwigo 2.2:

  • [feature 2260] Keep only two states for plugins (active and inactive)
  • [feature 2271] Merge autoupdate plugin into piwigo core.
  • [feature 2284] Create a class to manipulate images.
  • [feature 2259] Ability to resize the thumbnails and websize images
  • [feature 2274] Create thumbnails through ajax
  • [feature 2273] Ability to crop thumbnail (fixed size)
  • ability to use the command line Image Magick tools instead of PHP Imagick or GD libraries
  • [bug 2224] Unable to change images per page.
  • [bug 1874] The template "month_calendar.tpl" not found in templates list (models)

2) Screenshots


The plugin manager gets a new compact design, and only two states : a plugin is active or inactive (this makes plugin management much simpler)




Plugin Piwigo Auto Upgrade was integrated into Piwigo core as a standard feature




Ability to crop thumbnails, you can create square thumbnails for example, no need to install a plugin




Menubar is visible on all pages (tags, profile, comments, registration)

3) Start to beta test

Download Piwigo 2.3.0beta1

Please, try to open a new topic in the beta testing forum for each problem your encounter.

Piwigo 2.3.0beta2 will be released in one month, with new features. See planning details on [Piwigo Forums] Piwigo 2.3 planning.


Have a nice time while discovering Piwigo 2.3 new features!
