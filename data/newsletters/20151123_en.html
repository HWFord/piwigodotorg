Piwigo moves to Github

After years of hosting our own SVN (subversion) repository on piwigo.org server, here comes a major change for our coding activities. Moving to Github is a major shift, led and prepared by Mistic for months.



Support us on Github: show that you like Piwigo by starring us!

Read details and star us on Github


Piwigo Remote Sync

Piwigo Remote Sync is a new uploader for Piwigo, created by Mhelleboid. Piwigo Remote Sync will automatically create albums based on your folders. You can run the synchronization many times: Piwigo Remote Sync will only upload new albums and photos.



Piwigo Remote Sync works on Windows, Mac and Linux. Piwigo Remote Sync is available in alpha version (current version 0.0.9). We need you to alpha-test it.

Alpha-test Piwigo Remote Sync


MacSharePiwigo

After the iPhoto and the Aperture export plugins, Marcsins strikes back with MacSharePiwigo, a new uploader for MacOSX users. MacSharePiwigo comes as a "sharing module".



MacSharePiwigo is in beta-test, we need your feedback to make it as robust as possible.

Beta-test MacSharePiwigo


Testimonials wanted

When searching for a new application, we always wonder what users say about it. Unfortunately Piwigo doesn't have such testimonials on piwigo.org. We need yours :-) Tell us what you like in your Piwigo, how easy it is to install and use it, which problem it solved for you, how fast it is, how customizable it is... you get the idea!

We need all kinds of users here: families, amateur photographers, schools, photo clubs, pro photographers, companies... Short or long, your testimonial will help us to make Piwigo even more popular!

We will publish testimonials on a dedicated page, with your name, the publication date and a link to your Piwigo (or not, if you have only private content).
