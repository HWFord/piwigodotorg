Hello Piwigo user, you are receiving this email because you have signed up to the Piwigo Newsletter. Piwigo is the Free Open Source photo gallery software for the web.

Piwigo 2.3.4

Piwigo 2.3.4 is available. A few security bugs are fixed in this version, and even if we don't give full details, read Piwigo 2.3.4 Release Notes.

The Finnish language was added by Jussi Sunnari and Tome-. Piwigo is now available in 47 languages!

Follow the automatic upgrade procedure and run the most recent version of Piwigo.

Test version 2.4

Piwigo 2.4 is now under active test and we're coding last changes. Current version is 2.4.0RC2 (RC = Release Candidate = unstable version). On the following links you can learn more about all changes introduced in Piwigo 2.4:
  • announcement for 2.4.0beta1
  • announcement for 2.4.0beta2
  • announcement for 2.4.0RC1
  • announcement for 2.4.0RC2
Here is a summary of big changes compared to Piwigo 2.3:
  • multiple size: for each photo, Piwigo generates 9 sizes (square, thumbnail, XXS, XS, S, M, L, XL, XXL)
  • new default theme Elegant, modern and simple
  • mobile theme for smartphones (iphone, android...)
  • improved user interface on administration panel


The new default theme "Elegant" includes a sliding pane for photo properties.

We need your tests and feedback to make Piwigo 2.4.0 as rock solid as possible!

How to test? Test Piwigo 2.4 live demo or Download Piwigo 2.4.0RC2
