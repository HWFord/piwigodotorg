Piwigo 2.3.0RC1, are you ready to test?

As planned in our roadmap for Piwigo 2.3, now comes the time for release candidates with the first of them: Piwigo 2.3.0RC1. This version is for test only, don't install it on your live website.

For those who have followed the beta versions of the previous months, you already know that Piwigo 2.3 brings many new features. For others, you can make up for lost time by reading the previous announcements:

1) Changes

Here follows the list of changes available in version 2.3.0RC1, compared to version 2.3.0beta3:
  • [bug 2426] admins shouldn't be able to perform any action on plugins
  • [feature 1729] Use the picture name instead of file name in the title attribut
  • [feature 2427] update jQuery to version 1.6.2
  • [feature 1078] ability to merge tags
  • [feature 2418] Ability to give the same album permissions to sub albums
  • [feature 2420] automatically use the gallery title in the page banner
  • [feature 2417] add upload progression details and progress bar
  • [feature 1797] thumbnail ALT attribute:use photo title instead of filename
  • [feature 2027] redesign the lost password feature
  • [feature 2413] move gallery_url configuration parameter from database to local configuration file
  • [bug 1653] Upload failed whithout error when image size exeed upload_max_filesize
  • [feature 2410] upgrade to uploadify 3.0.0
  • [bug 2411] the "Browse" button is only translated in English
  • [bug 2412] ability to select GIF files with the Flash Uploader
  • [feature 2409] hide permissions by default
  • [feature 2407] display upload limitations before file selection
  • [feature 2408] change term "old style form" into "browser uploader"
  • [feature 2405] display uploaded photos in "real time"
  • [feature 2406] create new album before starting the upload
  • [bug 2373] Multiple file upload blocked by an error on any single file
2) Screenshots



The upload form was seriously improved. Among visible improvements, you will find a simpler album selector, maximum filesize and dimensions displayed before you select some photos, a selection button translated in all languages, a progress bar for the whole upload and last but not least, the photos are displayed as soon as they are uploaded, progressively.





The tooltip on thumbnail has been fully redesigned, just like the alternate text ("alt" html attribute), in order to improve search engine optimization and to provide more useful information to your visitors.





The gallery title can be used automatically in the page banner. Easy and convenient.





The "lost password" feature has been rewritten from scratch. The new feature let administrators reset their password without touching directly to the database.





You can merge tags... without losing associations with photos.

Translations on Piwigo.org

We're proud to present a new tool for translations. It was created by mistic100 who recently joined Piwigo Team. The main goal of this tool is to simplify the huge work of translation. Piwigo is currently translated in 39 languages. From now, translators can translate new sentences directly from their web browser.



This tool is available on piwigo.org/translate. It is currently in beta-test.
