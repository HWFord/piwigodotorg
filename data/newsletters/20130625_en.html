Hello Piwigo user, you are receiving this email because you have signed up to the Piwigo Newsletter. Piwigo is the Free Open Source photo gallery software for the web.

Piwigo 2.5.2

Piwigo version 2.5.2 is available, for a fresh install and for upgrade. Thanks to reporters and work achieved by developers, this version fixes many bugs, see full details on the Piwigo 2.5.2 Release Notes.

Each new release shows how active the translation team is:
  • 1 new language: Mongolian, learn more about Mongolia
  • 18 languages have been updated
Follow the automatic upgrade procedure and run the most recent version of Piwigo.

Piwigo 2.6 beta

Piwigo 2.6 is under active development. Here is the first beta version 2.6.0beta1. This version is for test only, so don't install it on your live website. Examples of change compared to Piwigo 2.5:
  • many more icons ("retina" display compatible) in administration
  • recent photos and recent albums are never empty
  • New layout for the group management page
  • upgrade template engine to Smarty 3

Many administration tabs and links get icons too.

Read more details and testing instructions on Piwigo 2.6.0beta1 announcement. Happy 2.6 beta testing!

Piwigo on Windows Phone

We're proud to let you know that a Piwigo application is available for Windows Phone. It was built by the Kwengi team! Piwigo for Windows Phone comes with numerous features:
  • select several photos at once
  • upload photos
  • display photos from albums
  • create albums and sub-albums
  • create tags
  • set title and tags on photos
  • progress bar on upload
  • directly delete photos from Piwigo
