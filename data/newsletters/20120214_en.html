Hello Piwigo user, you are receiving this email because you have signed up to the Piwigo Newsletter. Piwigo is the Free Open Source photo gallery software for the web.

Piwigo 2.3.3

Piwigo 2.3.3 is available. A few minor bugs are fixed in this version, see full details on the Piwigo 2.3.3 Release Notes.

The Esperanto was added by Axel Rousseau. Esperanto aims to be the "international language". Piwigo is now available in 46 languages!

And also a few additional features:

* a new screen to control rates by users (basically we needed it to avoid cheating on the Showcase)
* new option on API method pwg.images.add to update an existing photo

Follow the automatic upgrade procedure and run the most recent version of Piwigo.

Have a nice upgrade!

Piwigo 2.4 beta

Piwigo 2.4 is under active preparation. The first stable version is planned for April and for now we distribute beta versions. Keep in mind that Piwigo 2.4 beta versions are for test only, but we need your tests and feedback to make Piwigo 2.4.0 as rock solid as possible!

Here follow a few screenshots:



Multiple Size is the big new feature of Piwigo 2.4: once the "original" photo is uploaded, Piwigo generates "on the fly" several sizes for each photo.





Multiple Size on the picture page: you can select which size Piwigo displays.





Multiple size : you can select the photo size to display on the index page.





In the Batch Manager you can zoom on any photo without losing current selection and filter.





When registering a new user, Piwigo can send the connection settings by email.





The administration screen for an album has been redesigned for a simpler user interface. Actions such as photos sort order or permissions are dispatched in several tabs.





On the main page, the selector for the photo sort order was redesigned into an icon + link box.

Of course there are many more improvements in Piwigo 2.4, read 2.4.0beta1 and 2.4.0beta2 announcements for details.

How to start beta testing? Download Piwigo 2.4.0beta2, install it on your test environment and report bugs or feedback. Please, try to open a new topic in the beta testing forum for each problem you may encounter.

2011 in review and expectations for 2012

The Piwigo Team wishes you a happy new year 2012!

What happened in 2011?
  • 2 major versions 2.2 and 2.3, 8 minor versions and 9 beta versions were released. The release cycle was shortened to 6 months. An increased coding activity with 4,400 commits in the source code manager. 11 new languages: Piwigo now supports 45 languages.
  • a page for Piwigo on Google+ to publish news about Piwigo on this new social media and let our fans discuss about Piwigo.
  • applications Piwigo for iOS (iPhone/iPad) and Piwigo for Android: share your photos from your smartphone with simplicity
  • the Piwigo Showcase was born this summer. It currently gathers more than 300 examples of Piwigo galleries. The best galleries are tagged as Featured and 2 of them are randomly shown on piwigo.org home page.
  • mistic100 has joined the team and has quickly found his feet as developer
  • launch of an online translation plateform on piwigo.org/translate, created by mistic100
  • donations to the Piwigo Association have covered our hosting fees, domain names and various licenses required for coding and distribution. If you haven't done it yet, be aware that your donation is welcome, that it is guaranteed painless and that it will be really useful for us.
What can you expect for 2012? (at least regarding Piwigo...)
  • (for sure) export plugin for iPhoto
  • (for sure) export photos from Aperture
  • (for sure) Piwigo 2.4 during spring with important changes such as multiple size and a new default theme. Piwigo 2.5 expected for fall.
  • (for sure) small redesign on piwigo.org website with new pages: team members, logos and graphics, desktop and mobile applications, testimonials, more ideas for participation, a press review, a real blog instead of the forum for announcements...
  • (for sure) Zaphod joins Piwigo team. Zaphod has created theme Stripped.
  • (very likely) opening membership to the Piwigo Association
  • (likely) ebook about Piwigo
  • (likely) a basket plugin that let your visitors select some photos in order to download them or send them to a printing service
  • (dreamed) an export plugin for Picasa

As you can read the year 2012 looks promising, at least you are warned!
