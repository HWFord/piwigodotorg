Bonjour utilisateur de Piwigo, vous recevez cet newsletter car vous vous êtes enregistré pour recevoir des emails de la part du projet Piwigo. Piwigo est le logiciel libre de création de galerie photo pour le web.

Piwigo 2.2.2

La version 2.2.1 est disponible. Le Grec a été ajouté et plusieurs autres langues ont été mises à jour, plusieurs bugs ont été corrigés et quelques fonctionnalités mineures ont été ajoutées. Tous les détails sur les notes de la version 2.2.2

  • en raison de nombreux bugs sur le gestionnaire de tags, nous avons remplacé la bibliothèque technique. Le visuel est identique mais tous les bugs devraient être corrigés maintenant



  • sur la Gestion par lot, en mode unitaire, vous pouvez zoomer sur chaque photo avec une "lightbox" (plus facile de tagger une photo quand vous pouvez la voir en grand)

  • si vous utilisez la méthode d'ajout de photo par FTP + synchronisation et si vous avez de nombreux albums (des milliers d'albums), vous apprécierez le nouvel icône "synchroniser" sur l'écran d'administration de l'album : plus besoin de trouver l'album dans une liste de sélection sur l'écran de synchronisation,  un clic sur l'icône suffit.



  • De nouvelles options pour l'API Web : pwg.categories.getList est capable de retourner l'arborescence des albums à la place d'une liste plate et de pwg.categories.getImages renvoie désormais la date de création et la date d'ajout de chaque photo (pour plus de performances dans le plugin Lightroom de publication vers Piwigo)

Suivez la procédure de mise à jour automatique et utilisez la dernière version de Piwigo.

Piwigo 2.2.3

La version 2.2.3 a été publiée seulement quelques jours après la 2.2.2. Elle corrige des bugs introduits dans la version 2.2.2, tous les détails sont disponibles sur les notes de la version 2.2.3.

Piwigo 2.3.0beta1

Voici la version 2.3.0beta1. Cette version est seulement pour des tests, il ne faut pas l'installer sur votre vrai site. Même si Piwigo 2.2 a été publié il y a seulement quelques semaines, nous avons déjà de nouvelles fonctionnalités intéressantes à vous montrer pour la future version Piwigo 2.3.

1) Changements

Voici la liste des modifications par rapport à Piwigo 2.2 :

  • [fonctionnalité 2260] seulement 2 états pour les plugins (activé ou désactivé)
  • [fonctionnalité 2271] intégrer la mise à jour automatique en standard
  • [fonctionnalité 2284] une class PHP pour manipuler les photos
  • [fonctionnalité 2259] pouvoir redimensionner les miniatures et les tailles web déjà en ligne
  • [fonctionnalité 2274] génération des miniatures en AJAX (processus asynchrone)
  • [fonctionnalité 2273] rogner les miiatures (avec une taille fixe)
  • ability to use the command line Image Magick tools instead of PHP Imagick or GD libraries
  • [bug 2224] impossible de changer le nombre de miniatures par page
  • [bug 1874] le template "month_calendar.tpl" est introuvable dans la liste des modèles


2) Captures d'écran


Mise à jour du gestionnaire de plugin avec un nouveau design compact et simplification uniquement 2 états: un plugin est activé ou désactivé (ce qui rend la gestion des plugins beaucoup plus simple !)




Le plugin Piwigo Auto Upgrade a été intégré dans le noyau Piwigo


Possibilité de générer des vignettes carrées sans installation de plugin




Possibilité d'avoir Les menus sur toutes les pages (tags, le profil, les commentaires, enregistrement)
