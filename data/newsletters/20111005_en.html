Piwigo 2.3.0

Piwigo 2.3 is the new major version and is available for download or upgrade! You can read the Full release notes of Piwigo 2.3.0, which includes many screenshots and even a 3 minutes video! Here is the summary.

Piwigo 2.3 introduces new features like rating score, update manager and thumbnail cropping. There are enhancements to the user interface with a streamlined plugin manager and a more intelligent upload form. We have worked on keeping as many plugins and themes compatible with this new major version. Several new features that were available as plugins are now integrated into the Piwigo core.


Piwigo 2.3 in numbers: 50 new features, 600 code changes on Subversion, 6 new languages and a much shorter release cycle of 6 months.

[beta test] Piwigo for iOS : iPhone and iPad

P@t has started to code a new application for iOS (iPhone/iPad). It currently lets you upload one or several photos in your Piwigo photo gallery.


We need some beta-testers before we distribute the application on Apple application store. To become a beta-tester, we need your UDID that you can send to iosbeta@piwigo.org thanks to UDID Sender application. P@t will then send you the application and you will be able to install it with iTunes.

Piwigo for Android

Equivalent to Piwigo for iOS, Piwigo for Android is already available on Android Market.
