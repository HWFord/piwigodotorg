Hello Piwigo user, here comes fresh news about the Piwigo project.

Piwigo 2.6.3

Piwigo 2.6.3 is out. In addition to 2 minor issues, this stable version fixes a serious security bug, reported and fixed by Christopher Chrapka from ojezu.org. Update as soon as possible. Full details on the Piwigo 2.6.3 Release Notes

Follow the automatic upgrade procedure and run the most recent version of Piwigo.

Have a nice upgrade!

Piwigo 2.7, beta-testers needed!

Piwigo 2.7 is coming. Let's start with the first beta version. As usual with beta versions and release candidates, this version is for test only, so don't install it on your live website. Current improvements include:
  • HTML5 upload (see screenshot below)
  • search filter on photo batch manager
  • improved quick search
  • caches to increase speed performances
  • ability to change the photo time
  • and much more, see Piwigo 2.7.0beta1 announcement

HTML5 upload: no more Flash required, compatible with iPad/iPhone, drag and drop from your file browser, chunked upload...

We need your user feedback! We can provide 2.7 beta-testing installation of version 2.7, just ask :-).
